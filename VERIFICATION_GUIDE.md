# Fresh Bowl 웹사이트 검증 가이드 ✅

배포된 프로덕션 사이트의 모든 기능을 체계적으로 테스트하기 위한 가이드입니다.

---

## 📋 검증 체크리스트

### 1. 메인 페이지 (/)
**URL**: https://fresh-bowl.vercel.app

**확인 사항:**
- [ ] 페이지가 정상적으로 로드됨
- [ ] Fresh Bowl 히어로 섹션 표시
- [ ] "신선함이 살아있는 프리미엄 샐러드" 타이틀 표시
- [ ] "메뉴 둘러보기" 버튼 작동
- [ ] AI 추천 섹션 표시
- [ ] 인기 메뉴 3개 카드 표시 (그린 파워, 프로틴 부스트, 지중해)
- [ ] 각 상품 카드에 이미지, 이름, 가격, 칼로리, 단백질 정보 표시
- [ ] "담기" 버튼 작동
- [ ] Footer 링크 모두 표시

**예상 결과**: 깔끔한 green (#4ADE80) 테마의 랜딩 페이지

- [ ] 상품 클릭 시 상세 페이지로 이동
- [ ] 카테고리 필터 작동 (signature, classic, vegan)

**예상 제품:**
- 베지테리안 딜라이트

## 🚨 알려진 이슈

### 1. 이메일 인증 리다이렉트
**문제**: 회원가입 후 이메일 인증 링크가 localhost로 리다이렉트
**해결**: `SUPABASE_REDIRECT_FIX.md` 참고하여 Supabase 설정 변경

### 2. Supabase 데이터베이스
**상태**: 샘플 데이터 미입력 가능성
**해결**: `SUPABASE_SETUP.md` 참고하여 schema.sql 실행

### 3. OAuth 소셜 로그인
**상태**: Google/Kakao OAuth 미설정 가능성
**해결**: `OAUTH_SETUP.md` 참고하여 프로바이더 설정
